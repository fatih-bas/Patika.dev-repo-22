## React - Redux  NotesApp


### Features
- Create React App
- Redux Toolkit
- Chakra-ui

### Live

https://notes-app-111.netlify.app/

### Setup
1. Clone the repository and install the dependencies
```bash
Npm install or yarn install
```
2. Start the frontend application locally
```bash
Npm start or yarn start
```
